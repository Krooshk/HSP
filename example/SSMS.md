Categories - Основные категории продуктов питания
Contacts - Контакты с подробной информацией, ФИО, место проживания, область работы
CustomerCustomerDemo - пустая
CustomerDemographics - пустая
Customers - Похожа на контакты, можно сказать контакты клиентов
Employees - Работники, фото, дата рождения, дата найма, краткая сводка о человеке
EmployeeTerritories - Связь id работника с территорией
OrderDetails - детали заказа (id, цена, количество, скидка)
Orders - Заказы, id клиента, id работника, дата заказа, дедлайн по заказу, дата доставки, адрес доставки, им получателя 
Products - Продукты с названием, поставщиков id, категорией id, размер продукта, цена, количество на складе, количество в заказе, минимальное значение товаров, скидка
Region - 4 региона(части света)
Shippers - 3 логистических компании
Suppliers - компания, контактное имя, адрес, 
Territories - территории, связанные с регионом.
